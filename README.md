# 📊 Customer Churn Analysis

## Overview
Analyzed 7043 telecom customers to find why they leave.

## Tools Used
- Python, Pandas, NumPy
- Matplotlib, Seaborn
- Google Colab

## Key Findings
1. 26.54% overall churn rate
2. Month-to-Month contracts churn the most
3. Fiber Optic users churn more
4. New customers (0-12 months) leave the most
5. High monthly charges lead to more churn

## Dataset
Telco Customer Churn - Kaggle
